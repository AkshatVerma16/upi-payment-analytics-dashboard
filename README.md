# 🚀 UPI Payment Behavior & Fraud Analysis Dashboard

### 📌 Project Overview
A Fintech-focused data analytics project to analyze UPI transaction patterns, user segments, and potential fraud activities using Python and Streamlit.

### 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Plotly
- **GUI Framework:** Streamlit
- **Data Source:** Synthetic UPI Dataset (5000+ records)

### 🔍 Key Features
- **Time Analysis:** Identified peak transaction hours (e.g., 8 PM - 10 PM).
- **User Segmentation:** Classified users into Power, Regular, and Rare categories.
- **Fraud Detection:** Rule-based engine to flag high-velocity/high-value transactions.
- **Interactive GUI:** Dynamic filters for City and Merchant-based analysis.

### 🚦 How to Run
1. Clone the repository.
2. Install dependencies: `pip install pandas streamlit plotly`
3. Run the app: `streamlit run src/app.py`
